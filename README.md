# Proyek Tugas MySQL (INSERT & SELECT)

Proyek ini merupakan bagian dari tugas Pemrograman Web 2025-1.  
Fitur yang disediakan:

## ✔ Fitur Utama
- Membuat database `IF_NIM`
- Membuat tabel `user`
- Form registrasi user (Insert)
- Menampilkan data user (Select)
- Koneksi database menggunakan PHP MySQLi
